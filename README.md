# NestJs MySQL TypeORM Sample

```
{
    "type": "mysql",
    "host": "localhost",
    "port": 3306,
    "username": "root",
    "password": "root",
    "database": "test",
    "entities": ["dist/**/*.entity{.ts,.js}"],
    "synchronize": true
}
```
Then please install all the app depencencies by executing the following command:
```
npm install
```
and you can run the app by executing the following command:
```
npm run start:dev
```
